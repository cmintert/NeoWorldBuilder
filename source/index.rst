NeoWorldBuilder documentation
=============================

NeoWorldBuilder is a sophisticated worldbuilding tool that uses Neo4j graph database to organize creative ideas around four fundamental concepts: Names, Descriptions, Labels, and Relations.

Features
--------

* **Graph Database**: Uses Neo4j for efficient relationship management
* **Service-Oriented Architecture**: Clean separation of concerns
* **PyQt6 Interface**: Modern desktop application
* **AI Integration**: LLM-powered content enhancement
* **Map Component**: Interactive worldbuilding maps
* **Multi-Project Support**: Isolated project environments
